## 第一讲：百年未有之大变局<a id = "第一讲"></a>
解题：
- “现代”的表现特征：高楼大厦，汽车飞机，坦克大炮，蛋白质摄入，寿命提高，现代艺术。
- “现代”的经济学视角：经济结构的变化：减少传统农业部门的重要性，建成发达的资本主义社会
- 生产方式变化：工业化
- 能源结构的变化：无生物能源和使用工具增加使用效果
- “现代”的政治学视角：
   - 政治能力的提高：提取、调整、分配、象征、回应
   - 专门化：技术治理主义
   - 科层化
- “现代”的社会学视角：社会结构分化，社会体系中心化，机械技术、理性或世俗的态度，世界体系的形成（球化进程）
- “现代”的历史视角：
   - 元素改变：客观时间（格林尼治时间，标准时间）、多种群体、新概念
   - “多元现代”：世界各国走向现代化的路径不是一致的。（eg.中国式现代化既有各国代化的共同特征，还具有中国特色）
   - “叙事”：                 
      - 线性叙事：进步、发展
      - 近现代史的三种常见叙事（三种叙事统一来看）：
         - 国族构建与独立（中国人民寻求独立）
         - 社会经济的变化（中国经济从农业到工业）
         - 阶级斗争和革命（阶级斗争从资产阶级到无产阶级）

近现代史vs一般的历史
关注“长十九世纪”、“长二十世纪（一个革命的世纪）”
- 目的性：历史本体的考察vs“兴亡”“资鉴”
- 理性、科学性：社会发展规律vs“五德始终”
- 实践意义：获得共识

中国近现代史的本体
- 分界线：1840、1949两个年份
- 近代史：从1840 年鸦片战争爆发，到1949年中华人民共和国成立
- 现代史：从10949年建国至今

数千年未有之大变局解题：“数千年未有的变局”指的是中国从近代以来，因西方侵略而诞生的民族危机，鸦片战争事中国近代史的开端，是中国沦为半殖民地半封建社会的开始。第二次鸦片战争加速了中半殖民地半封建社会的进程，《马关条约》极大的加深了中国的半殖民地半封建的程度。《辛丑条约》标志着中国半殖民地半封建社会最终形成。   
如何认识变局？
一、鸦片战争前夜
1.1西方的扩张：工业革命，殖民扩张
世界：
- 15世纪末地理大发现，彻底改变了大西洋世界的政治地理，也改变了世界史
- 西班牙、法国：1500-1600年
- 1763年，英国取代法国，基本独占印度

经济来源：三角贸易（黑奴贸易）
第一阶段：殖民扩张：在资本原始积累期间大都采取赤裸裸的暴力手段（eg.武装占领，海外移民，海盗的掠夺，欺诈性的贸易，血腥的奴隶买卖）
科学革命：
- 16-17世纪的天文学
- 17-18世纪经典力学
- 19世纪前期化学

工业革命：
- 动力织布机
- 蒸汽机
- 火车
- 化学工业

大众消费主义：18世纪英法国内面对中低收入群体的消费现象，使得各阶级开始购买他们从未有机会购买的、甚至比以前更大范围的商品（Simmel理论，认为时尚本质上是一种模仿，中低等阶级通过购买小贵族能消费得起的奢侈品从而让自己置身贵族行列。）
第二阶段：工业革命后世界市场的扩张：
“帝国主义”（商品倾销地和原料掠夺地）
扩张的西方形成：
- 1640年英国资产阶级革命
- 1776年美国独立
- 1789年法国大革命

鸦片战争前夜的中国：
1.人口膨胀，环境压力激增（从宋至清持续增长，约1750年-1775年达到最佳状态即人均经济效益最大）
2.政治专制腐败
- 皇权不受制约，其代理人权力过度膨胀酿成全国性政治腐败
- 皇帝树立绝对权威
- 文化专制禁锢了中国人思想
3.自然经济：小农经济
- 以个体家庭为单位并与家庭手工业
- 自给自足的自然经济占主要地位跨地区、跨国贸易份额较小
- 小农经济的特点：高度集约化，高土地生产率，低劳动生产率；在面临风险时向有雇佣劳动而无资本化的“半无产化”转变
4.社会危机频发
- 1796年川楚白莲教起义
- 1813年河南、山东、直隶之天理教、八卦教起义
5.对外封闭：广州贸易，（一口通商的闭关锁国政策）
- 18世纪中后期成为中英主要贸易地点，“一口通商”
- 粤海关对广州贸易的控制：
   - 直接掌控征税权和管理权
   - 钳制华洋中介（引水人、通事、买办、行商等）
   - 洋商行动受限，情报不及时
- 1830年代的变化：
   - 英国散商逐渐团结
   - 蒸汽船的使用逐渐频繁
6.英国两次来华使团
- 1792年马嘎尔尼使团访华，得见乾隆皇帝，但所有要求均被拒绝。
- 1816年阿美士德师团访华，未能得见嘉庆皇帝

鸦片战争前夜的中英贸易
- 英国资本主义快速发展
- 18世纪中叶开始的工业革命极大的改变经济结构和社会面貌
- 棉纺织业的蓬勃发展，需要在全国打通销路
- 需求激增
- 对中国而言，与工业品相比，白银的地位更重要。
- 对于中欧贸易而言，既是商品也是货币。
- 1810年代的危机：拉丁美洲独立运动。

中英贸易的基本形态：三角贸易
- 英方目的：
   - 减少中美贸易中白银的支付量
   - 提升棉纺织品的出口量
   - 掌控印度的经济
- 实施方法：
   - 英国向印度销售棉纺织品及其他工业制品
   - 英国商人从印度向中国出口鸦片
   - 出售后的盈余则在中国购买茶叶等货物。
   - 伦敦和加尔各答的金融结算体系为支撑。
- 英方力图扩大中国市场，认定广州贸易体制是掣肘因素。

鸦片贸易扩大
- 鸦片贸易在19世纪前期迅速扩大
- 华南、华中的国内跨区域鸦片贸易被逐级打开，需求暴涨
- 鸦片贸易带来的经济和社会问题迅速凸显

鸦片的危害：
- 危害人民身体健康
- 白银外流
- 吏治腐败，军队腐化

禁烟、销烟
- 雍正七年（1729年）禁令：禁止贩售和吸食
- 嘉庆朝三次申令：1799年，1807年，1809年。
- 1836年，许乃济奏请废止禁令，准许种植罂粟，遭到强烈反对，清廷重申禁令。
- 1839年3月，林则徐到达广州，着手收缴和销毁鸦片。
- 1840年7月4日，英国远征舰队炮击定海。

结论：鸦片贸易与近代帝国主义
- 鸦片贸易是中国进入近代史时面临的第一次重大危机
- 这不是一两个野心家的政府大业，也不是一两个军事集团的战争狂热，而是近代西方帝国主义掌控下世界秩序队对中国的初次进攻。

小结：
- 欧美资产阶级政权为资本主义的发展创造和提供了政治上的前提和保证。18世纪中叶至19世纪中叶，从英国开始然后迅速推广到欧美各国的工业革命，使资本主义经济得到迅速发展。
- 殖民主义适应西方资本主义的发展要求而产生，19世纪末，资本主义进入帝国主义阶段，进一步发展成为由少数帝国主义强国主宰的更完整的世界体系。.18世纪后期开始，史无前例的人口增长使中国社会负担和环境压力骤增。经济模式落后，人民生活水平增长停滞。政治上，强大的中央集权政府无力处置严重的腐败和低效问题。思想文化禁锢。军备废弛。
- 鸦片贸易令中国面临巨大危机，成为中英关系的导火索，英方挑起侵略战争。

## 第二讲：帝国主义对中国的侵略<a id = "第二讲"></a>

（五）自然经济的瓦解
- 鸦片战争后我国农村自然经济的加速分解，主要不是由于生产力的发展和相应社会分工造成的，而是由于资本主义列强的商品入侵和出口贸易的需求造成的，是一种被动的，强制的分解。
- 这种分解，虽然有利于商品经济的扩大，但它未能触动封建土地所有制，未能改变小农经济的生产方式，并且由于这种分解所形成的商品市场是一种半殖民地型的市场，商品量和价格水平都受国际市场支配。
- 农村小生产者在封建剥削之外，又加上了资本主义剥削。

（二）传教士在中国的活动
- 19世纪60年代后，外国传教士大批来到中国，并进入内地，边疆和少数民族地区。

（六）经济与技术落后

（七）社会制度腐败 
政治腐败：宫廷
- 冲主在位，皇权不振
   - 穆宗载淳：7岁即位，18岁亲政，不到19岁死亡
   - 德宗载湉：3岁即位，18岁亲政，27岁被囚禁。
- 慈禧太后专权
   - 辛酉政变(1861)：处死肃顺。
   - 甲申易枢(1884)：罢免奕訢。
   - 戊戌政变(1898)：囚禁光绪帝。
   - 庚子易储(1900)：立大阿哥溥。
- 宫廷政治掣肘国家治理
   - 国家治理：理性，建设性，从全局出发，为国家和人民谋利益
   - 宫廷政治：以慈禧太后及少数内廷要人一时喜怒为转移
   - 政治腐败：统治机器
- 中央层面
   - 八旗贵族+科举精英
	- 满汉关系问题时隐时现
   - 科举精英官场守旧风气	
- 省级层面
   - "局员"：非正式官僚体系。
   - 外销钱粮：各省的"小金库"
   - 分权但不能有效协作。
   - 无法推动大局变革。

（八）基层治理失灵
- 士绅权力逐渐扩大
   - 太平天国运动以后，士绅自治组织逐渐成为官府与普通民众之间的重要力量
   - 士绅群体哺育了一部分近代社会，又因把持乡村社会而成为更大范围变革的阻碍变革；
   - 士绅组织背后的传统习俗伦理与现代观念相冲突。
- 但有官民之别，无人民主权意识。
   - 官府视人民为管治和榨取对象。
   - 阶级压迫，性别压迫，族群压迫。
   - 动员能力很低
   - 无视人民中间蕴含的伟大力量
- 压榨人民

（九）后发现代化的命运
后发现代化：
- 时宜性概念，启动较晚
- 外源性/传导性
- 最初动力来源于外部

防卫型现代化：
- 防卫性目的：改变中国落后挨打的局面
- 防卫性措施：优先针对外国侵略危机较严重的领域
- 防卫性评估：以对抗外国侵略为标准
